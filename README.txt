# V&G Logistic

Oficjalna strona internetowa firmy V&G Logistic działającej na GrandRP.

## Co znajduje się na stronie?

- informacje o firmie,
- aktualności i rozwój V&G Logistic,
- zarząd firmy,
- flota,
- partnerzy,
- rekrutacja,
- regulamin IC/OOC,
- link do Discorda.

## Aktualizacja

Strona zawiera aktualne informacje:

- około 1000 kursów tygodniowo,
- 2 ciężarówki w firmie,
- 3 ciężarówki w drodze od dealera,
- dostawy części do mechanika z Czech,
- dostawy składników do restauracji,
- badania psychologiczne dla kandydatów na kierowców,
- pełny regulamin firmy bez przekierowania do Google Docs.

## GitHub Pages

Strona działa przez GitHub Pages.

Po każdej zmianie w plikach:

- `index.html`
- `style.css`
- `script.js`

wystarczy kliknąć **Commit changes**, a GitHub automatycznie zaktualizuje stronę.
