JAK WRZUCIĆ STRONĘ V&G LOGISTIC NA GITHUB PAGES

1. Wejdź na https://github.com i załóż konto lub zaloguj się.
2. Kliknij + w prawym górnym rogu -> New repository.
3. Nazwa repozytorium: vg-logistic
4. Ustaw Public.
5. Kliknij Create repository.
6. Wrzuć do repozytorium pliki:
   - index.html
   - style.css
   - script.js
7. Wejdź w Settings -> Pages.
8. W Source wybierz: Deploy from a branch.
9. Branch: main / root.
10. Kliknij Save.

Po chwili strona będzie dostępna pod adresem podobnym do:
https://twojanazwa.github.io/vg-logistic/

WAŻNE:
- W pliku index.html są dwa linki Discorda: https://discord.gg/
  Podmień je na prawdziwy link do Discorda V&G Logistic.
- Statystyki typu 100+ kursów możesz edytować ręcznie w index.html.
- Partnerów możesz zmienić w sekcji „Nasi partnerzy”.
